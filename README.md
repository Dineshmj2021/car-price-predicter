# 🚗 Car Price Prediction Using Machine Learning

## 📌 Project Overview

Car Price Prediction is a Machine Learning project that predicts the selling price of used cars based on various factors such as car age, present price, kilometers driven, fuel type, transmission type, seller type, and ownership history.

The project includes data preprocessing, statistical analysis, data visualization, model training using Linear Regression, and deployment through Streamlit.

---

## 🎯 Objectives

* Analyze used car data
* Perform statistical analysis
* Visualize important trends
* Build a machine learning model for price prediction
* Deploy the model using Streamlit

## 🚀 How to Run the Project

### Step 1: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 2: Train the Model

```bash
python train_model.py
```

This will generate:

```text
model.pkl
```

### Step 3: Run Streamlit Application

```bash
streamlit run app.py
```
